# Unity Toolbox

Коллекция инструментов для Unity Editor, призванная ускорить привычные рабочие процессы разработки и избавить от рутины. Все скрипты находятся в папке `Editor/` и доступны из следующих меню:

- `Tools/Megxlord Toolbox`
- `Tools/Megxlord uGUI`

# 2D

Папка: `Editor/2D/`

## ObjectSnap.cs

**Меню:** `Tools/Megxlord Toolbox/2D/Object Snap 2D`

Выравнивает все выбранные 2D-объекты по позиции основного выделения (последнего выбранного объекта). Основной объект остается на месте, в то время как все остальные выбранные объекты перемещаются на его координаты X и Y.

**Назначение:** Быстрое выравнивание спрайтов без необходимости копировать координаты вручную. Поддерживает Undo (отмену действия).

---

## Scale Calculator 2D.cs

**Меню:** `Tools/Megxlord Toolbox/2D/Scale Calculator 2D`

Продвинутый калькулятор масштабирования для 2D-спрайтов. Автоматически определяет режим отрисовки `SpriteRenderer` (`Simple`, `Sliced` или `Tiled`) и применяет соответствующий метод изменения размера.

### Возможности

- Пресеты соотношения сторон (1:1, 4:3, 16:9, 9:16 и т. д.)
- Режимы масштабирования:
  - По длинной стороне (Longest Side)
  - По короткой стороне (Shortest Side)
  - По ширине (Width)
  - По высоте (Height)
- Сохранение соотношения сторон (Keep Aspect Ratio)
- Автоприменение к выбранным объектам
- Округление до целых чисел
- Копирование/вставка размеров
- Восстановление исходных значений

**Назначение:** Точное изменение размеров спрайтов с сохранением пропорций. Полезно при подготовке ассетов для интерфейса (UI) и 2D-игр.

---

## Sprite2DOrderManager.cs

**Меню:** `Tools/Megxlord Toolbox/2D/Sprite2D Order Manager`

Менеджер порядка рендеринга 2D-объектов на уровне всей сцены.

Поддерживает:

- SpriteRenderer
- TextMeshPro
- MeshRenderer
- ParticleSystemRenderer
- TilemapRenderer
- SpriteMask
- LineRenderer
- TrailRenderer
- SkinnedMeshRenderer

### Возможности

- Таблица рендереров с:
  - Состоянием активности (Enabled)
  - Типом компонента
  - Именем объекта
  - Слоем сортировки (Sorting Layer)
  - Порядком в слое (Order in Layer)
  - Маской слоя рендеринга (Rendering Layer Mask)
  - Цветом
- Фильтрация по типу рендерера, поиск и фильтрация по слою сортировки
- Пакетное редактирование Sorting Layer, Order in Layer и цвета
- Обнаружение конфликтов рендеринга (перекрывающиеся рендереры с одинаковым значением Order in Layer)
- Визуализация порядка отрисовки
- Статистика сцены
- Включение/выключение GameObjects

**Назначение:** Эффективная организация порядка отрисовки в больших 2D-сценах и обнаружение конфликтов сортировки.

# 3D

Папка: `Editor/3D/`

## RandomScale.cs

**Меню:** `Tools/Megxlord Toolbox/3D/RandomScale`

Применяет случайный масштаб в заданном настраиваемом диапазоне ко всем выбранным объектам.

Идеально подходит для придания естественного разнообразия камням, деревьям, растительности и другим объектам окружения.

**Назначение:** Быстрое добавление визуального разнообразия без необходимости масштабировать каждый объект вручную.

---

## SnapToTerrain.cs

**Меню:** `Tools/Megxlord Toolbox/3D/Snap To Terrain`

Прижимает выбранные объекты к поверхности террейна (ландшафта).

Поддерживает два режима размещения:

- **Edge Pivot** — самая нижняя точка объекта касается террейна.
- **Center Pivot** — выравнивает центр объекта с учетом границ его меша.

Поддерживает английскую и русскую локализацию.

**Назначение:** Автоматическое размещение пропсов, таких как деревья, камни, здания и другие объекты, на террейне.

## Hotkeys.cs

**Меню:** `HotKey/`

Добавляет горячие клавиши для часто используемых операций в редакторе.

| Горячая клавиша | Действие |
|-----------------|----------|
| `B` | Запуск игры + Обновление ассетов (Play + Refresh Assets) |
| `Shift+B` | Переключение режима игры (Toggle Play Mode) |
| `F5` | Обновление ассетов (Refresh Assets) |
| `F12` | Создать папку (Create Folder) |
| `Shift+5` | Создать пустой C# скрипт (Create Empty C# Script) |
| `1` | Открыть настройки Input System |

**Назначение:** Сокращение рутинных действий в редакторе и ускорение ежедневного рабочего процесса.

---

## SceneSwitcherToolbar.cs

Добавляет два выпадающих списка на главную панель инструментов Unity.

### Режимы

1. Все сцены (All Scenes)
2. Рабочие сцены (Work Scenes)
3. Сцены в сборке (Scenes in Build)

### Возможности

- Быстрое переключение между сценами
- Фильтрация по настройкам сборки (Build Settings)
- Загрузка сцен во время выполнения в режиме игры (Play Mode)
- Автоматическое обновление при изменении ассетов проекта

**Назначение:** Мгновенное переключение между сценами без необходимости искать их в окне Project.

---

## SmartProjectMenu.cs

Предоставляет быстрый доступ к инструментам анализа рендеринга Unity:

- Frame Debugger
- Render Graph Viewer

**Назначение:** Доступ к диагностике рендеринга Unity в один клик.

# Иерархия

Папка: `Editor/Hierarchy/`

## MissingScriptValidator.cs

**Меню:** `Tools/Megxlord Toolbox/Hierarchy/Missing Script Validator`

Сканирует каждый GameObject в текущей сцене на наличие отсутствующих ссылок на скрипты (Missing Scripts).

Отображает:

- Имя объекта
- Путь в иерархии
- Количество отсутствующих компонентов
- Кнопку выбора (Select) для быстрого перехода к объекту

**Назначение:** Обнаружение сломанных ссылок на скрипты после рефакторинга, импорта ассетов или обновления версии проекта.

---

## Polycount Viewer.cs

**Меню:** `Tools/Megxlord Toolbox/Hierarchy/Polycount Viewer`

Отображает статистику полигонов и вершин для выбранных объектов.

### Возможности

- Количество вершин MeshFilter
- Количество вершин MeshCollider
- Общее количество полигонов для префабов (prefabs)
- Статистика LODGroup (включая LOD0)

**Назначение:** Быстрое выявление высокополигональных ассетов в процессе оптимизации.

# Режим игры (PlayMode)

Папка: `Editor/PlayMode/`

## TrueFullscreenGameView.cs

**Меню:** `Tools/Megxlord Toolbox/Fullscreen/Toggle Fullscreen GameView` (`F11`)

Предоставляет настоящий полноэкранный режим для Game View с использованием Windows API вместо встроенного в Unity режима максимизации окна.

### Возможности

- Реализация через Win32 API
- Стиль окна `WS_POPUP`
- Полноэкранный режим без рамок (Borderless fullscreen)
- Удаляет панель инструментов и отступы окна
- Выход по нажатию `Escape`
- Дополнительная опция автоматического развертывания на весь экран при входе в режим игры (Play Mode)

**Назначение:** Тестирование игр в настоящем полноэкранном режиме прямо внутри Unity Editor.

> **Примечание:** Только для редактора под Windows (`UNITY_EDITOR_WIN`).

# Скрипты

Папка: `Editor/Scripts/`

## ReferencingScriptsWindow.cs

**Меню:** `Tools/Megxlord Toolbox/Scripts/Find Referencing Scripts`

Находит каждый скрипт в проекте, который ссылается на выбранный класс.

Выполняет анализ исходного кода, игнорируя комментарии и строковые литералы.

Определяет:

- Наследование
- Инициализацию (создание экземпляров) объектов
- Объявление переменных
- Вызовы методов

**Назначение:** Анализ зависимостей перед рефакторингом или удалением класса.

---

## ScriptUsageFinder.cs

**Меню:** `Tools/Megxlord Toolbox/Scripts/Find Script Usage`

Ищет в текущей сцене и во всех префабах проекта компоненты выбранного скрипта.

Поддерживает перетаскивание (drag-and-drop) ассетов `MonoScript`.

**Назначение:** Поиск всех GameObject, использующих определенный компонент, перед его изменением или удалением.

---

## ScriptArchitecture.cs

**Меню:** `Tools/Megxlord Toolbox/Scripts/Script Architecture`

Продвинутый анализатор производительности скриптов.

Сканирует все файлы C# в папке `Assets/` и выявляет методы, выполняемые каждый кадр.

### Поддерживаемые колбэки

- Update
- LateUpdate
- FixedUpdate
- OnGUI
- OnAnimatorMove
- OnAnimatorIK
- OnRenderObject
- OnWillRenderObject

Каждый метод получает оценку нагрузки **Load Score (0–100)** на основе:

- Типа колбэка
- Размера метода
- Паттернов с рисками для производительности

### Флаги рисков

| Флаг | Описание |
|------|----------|
| `A` | Аллокации памяти (`new`, `Instantiate`, `GetComponent`) |
| `F` | Поиск по всей сцене (`FindObjectsOfType`, `GameObject.Find`) |
| `SM` | `SendMessage` / `BroadcastMessage` |
| `RC` | Запросы физики (`Raycast`, `OverlapSphere` и т. д.) |
| `LQ` | Использование LINQ |
| `N²` | Вложенные циклы |
| `LP` | Циклы |
| `MC` | Дополнительные вызовы методов |

### Возможности

- Параллельное сканирование
- Фильтрация
- Сортировка
- Рекомендации по оптимизации

**Назначение:** Выявление потенциальных узких мест в производительности до начала профилирования.

# uGUI

Папка: `Editor/uGUI/`

## AnchorsToCornersChecker.cs

**Меню:** `Tools/Megxlord uGUI/Anchors to Corners Checker`

Проверяет `RectTransform` на использование растянутых якорей (Stretch anchors).

Проверяет, что:

- `anchorMin` / `anchorMax` соответствуют визуальным границам
- Смещения (offsets) равны нулю

Опции:

- Игнорировать компоненты Text и TMP
- Проверявать фиксированные якоря
- Включать неактивные объекты

**Назначение:** Обнаружение неверно настроенных якорей интерфейса (UI).

---

## AnchorsToCornersEditor.cs

**Меню:** `Tools/Megxlord uGUI/Anchors to Corners`

Автоматически настраивает якоря в соответствии с текущими визуальными границами выбранных `RectTransform` и сбрасывает смещения в ноль.

Поддерживает множественный выбор.

**Назначение:** Преобразование существующих макетов интерфейса в правильно настроенные якоря в один клик.

---

## Scale Calculator.cs

**Меню:** `Tools/Megxlord uGUI/Scale Calculator`

Комплексная утилита для масштабирования интерфейса с четырьмя вкладками.

### Scale Calculator

- Свободное масштабирование (Free Scale)
- Масштабирование по ширине
- Масштабирование по высоте
- Вписать внутрь (Fit Inside)
- Заполнить снаружи (Fill Outside)
- Ползунок равномерного масштабирования (Uniform Scale)
- Быстрые кнопки ×2 / ×½ / ×3
- Применение размера или масштаба к выбранным `RectTransform`

### Ratio Calculator

- Расчет недостающих размеров
- Визуализация соотношения сторон
- Популярные соотношения сторон

### Resolution Calculator

- Конвертация между популярными разрешениями устройств
- Справочная таблица масштабирования UI

### Batch Scale

- Масштабирование нескольких объектов
- Использование `localScale` или `sizeDelta`
- Сохранение мировой позиции
- Предварительный просмотр изменений

**Назначение:** Изменение размеров и адаптация UI под различные разрешения экранов.

## UIRectValidatorWindow.cs

**Меню:** `Tools/Megxlord uGUI/UI Rect Validator`

Проверяет значения `RectTransform` по всей сцене.

Проверяет:

- Anchored Position

## FigmaLayoutEditor.cs

**Меню:** `Tools/Megxlord uGUI/Figma Layout Editor`

Визуальный редактор макетов интерфейса, вдохновленный ограничениями (constraints) в Figma.

Позволяет редактировать отступы (`Left`, `Right`, `Top`, `Bottom`) и размеры относительно родительского `RectTransform`.

### Вкладки

- Layout (Макет)
- Align (Выравнивание)
- Constraints (Ограничения)
- Settings (Настройки)

### Возможности

- Режим привязки к родителю или к Canvas
- Масштабирование и панорамирование
- Сетка и линейки
- Оверлей в реальном времени в окне Scene View
- Привязка к пикселям
- Привязка к сетке
- Визуализация безопасной зоны (Safe area)
- Интерактивные маркеры изменения размера

**Назначение:** Позиционирование элементов UI с использованием отступов и ограничений вместо ручного редактирования якорей и `anchoredPosition`, предлагая рабочий процесс, схожий с Figma.

# Установка

1. Скопируйте папку `Editor/` в директорию `Assets/` вашего проекта Unity.
2. Убедитесь, что файл `MyEditorScripts.asmdef` включен в проект.
3. Все инструменты появятся в меню:
   - `Tools/Megxlord Toolbox`
   - `Tools/Megxlord uGUI`

---

# Требования

- Unity 2021.3 или новее
- Некоторые функции используют метод `FindObjectsByType`, доступный в Unity 6.
- TextMeshPro (необязательно; требуется только для поддержки TMP в скрипте `Sprite2DOrderManager`).
